# Instructions
사용자 질문에서 폐기물 중분류를 추출합니다.

## 가능한 카테고리
- 플라스틱
- 종이류
- 캔류
- 유리류
- 비닐류
- 스티로폼
- 음식물쓰레기
- 소형가전
- 의류
- 일반쓰레기

---

# Input Format

```xml
<context id="user_message">
  사용자 질문
</context>
```

---

# Output Format
카테고리명만 출력 (예: "플라스틱")
추출할 수 없으면 "unknown" 출력

---

# Context Input

<context id="user_message">
{message}
</context>
