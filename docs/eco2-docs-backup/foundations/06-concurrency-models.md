## 1. 4к°Җм§Җ лҸҷмӢңм„ұ лӘЁлҚё

Pythonм—җм„ң мӮ¬мҡ© к°ҖлҠҘн•ң лҸҷмӢңм„ұ лӘЁлҚё:

| лӘЁлҚё | лӢЁмң„ | нҠ№м§• | мӮ¬мҡ© мӮ¬лЎҖ |
|------|------|------|----------|
| **Process** | н”„лЎңм„ёмҠӨ | мҷ„м „ кІ©лҰ¬, GIL мҡ°нҡҢ | CPU-bound |
| **Thread** | OS мҠӨл Ҳл“ң | кіөмң  л©”лӘЁлҰ¬, GIL мҳҒн–Ҙ | I/O-bound (м ңн•ңм Ғ) |

```python
from multiprocessing import Process, Queue

def worker(q, n):
    """CPU-intensive мһ‘м—…"""
    result = sum(i * i for i in range(n))
    q.put(result)

if __name__ == "__main__":
    q = Queue()
    processes = []
    
    for i in range(4):  # 4к°ң н”„лЎңм„ёмҠӨ мғқм„ұ
        p = Process(target=worker, args=(q, 1000000))
        p.start()
        processes.append(p)
    
    for p in processes:
        p.join()
    
    results = [q.get() for _ in range(4)]
```

```python
import threading
import time

counter = 0
lock = threading.Lock()

def worker():
    global counter
    for _ in range(100000):
        with lock:  # Race condition л°©м§Җ
            counter += 1

threads = [threading.Thread(target=worker) for _ in range(4)]
for t in threads:
    t.start()
for t in threads:
    t.join()

print(f"Counter: {counter}")  # 400000
```

### 4.6 Python мҳҲм ң (Gevent)

```python
import gevent
from gevent import monkey
monkey.patch_all()  # н‘ңмӨҖ лқјмқҙлёҢлҹ¬лҰ¬ нҢЁм№ҳ

import requests  # мқҙм ң л№„лҸҷкё°мІҳлҹј лҸҷмһ‘

def fetch(url):
    response = requests.get(url)  # мһҗлҸҷмңјлЎң greenlet м „нҷҳ
    return response.status_code

# лҸҷмӢңм—җ м—¬лҹ¬ мҡ”мІӯ
urls = [
    "https://api.github.com",
    "https://api.twitter.com",
    "https://api.facebook.com",
]

# gevent.spawn()мңјлЎң greenlet мғқм„ұ
greenlets = [gevent.spawn(fetch, url) for url in urls]
gevent.joinall(greenlets)

results = [g.value for g in greenlets]
```

            fetch(session, "https://api.github.com/orgs/python"),
        ]
        results = await asyncio.gather(*tasks)
        return results
